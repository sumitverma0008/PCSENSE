# PCSensei Android App

This directory will contain the Android application for PCSensei.

## Planned Features
- Mobile-friendly PC/Laptop recommendation wizard
- Price comparison across retailers
- Push notifications for price drops
- Offline mode with cached recommendations

## Tech Stack (Proposed)
- React Native / Flutter / Native Android (Kotlin)
- REST API integration with backend
- Local SQLite for offline caching
